1. To initialize the project using vite you use the command "npm create vite@latest ."
to install the vite in the current location hence the "." symbol

2.It will show you bunch of project that you can choose from and then install 
the vite into your favourite one.

3.you also need to choose between ts or js

4. vite will generate the placeholder codes to indicate it is working as well as some configurations

5. also we will need to use "npm i" to install the dependencies into the project

6. we also use the vite as the server by using "npm run dev" command and copy the url
into the web browser.
